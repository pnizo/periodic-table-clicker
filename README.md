# Periodic Table Clicker - 元素周期表クリッカー

周期表をモチーフにした放置/クリッカーゲーム。粒子を生成し、元素を合成して宇宙のエネルギーを最大化しよう！

## 🎮 プレイ方法

[GitHub Pagesでプレイ](https://pnizo.github.io/periodic-table-clicker/)

または、ローカルで `index.html` を開いてください。

## ✨ 特徴

- **クリッカーメカニクス**: エネルギーボタンをクリックしてエネルギーを生成
- **粒子生成機**: 電子、陽子、中性子の自動生成システム
- **元素合成**: 118種類の全元素を合成可能（核融合レシピ）
- **BigInt対応**: 非常に大きな数値も正確に処理
- **実績システム**: 340以上の実績（元素、周期、族、クリック数、プレイ時間、粒子数）
- **プレステージ（Re-universe）**: 総獲得エネルギーに応じた倍率ボーナス
- **インタラクティブUI**: ホバーでレシピ表示、色分けされた利用可能性表示
- **自動セーブ**: 10秒ごとに自動保存（localStorage使用）
- **複数タブ防止**: データ整合性のため同時に1タブのみ起動可能
- **モダンデザイン**: ダークテーマ、グラスモーフィズム、スムーズなアニメーション

## 🎯 ゲームの目標

1. エネルギーを生成して粒子生成機をアップグレード
2. 粒子を集めて元素を合成
3. より高度な元素を合成して宇宙エネルギーを最大化
4. 実績を解除してボーナス倍率を獲得
5. Re-universeで倍率を上げてさらに効率よくプレイ

## 🎨 機能詳細

### エネルギーシステム
- クリックで+1エネルギー（倍率適用）
- 粒子から自動生成: `(電子 + 陽子 + 中性子) × 0.1` e/秒
- 粒子保有数から自動生成: `保有数 / 60` e/秒
- 元素から自動生成: `元素数 × エネルギー値 × 0.05` e/秒
- 実績ボーナス: 解除した実績数 × 100% が全生成量に乗算

### 粒子生成機
| 粒子 | 初期コスト | 基本レート |
|------|-----------|-----------|
| 電子 | 10e | 1.0/秒 |
| 陽子 | 100e | 0.5/秒 |
| 中性子 | 1840e | 0.167/秒 |

- コスト倍率: 1.15^レベル
- 生成速度: 基本レート × レベル

### 元素合成
- 118種類の全元素を実装（H〜Og）
- 核融合レシピ: 軽い元素を組み合わせて重い元素を合成
- 一括合成: x1, x10, x100, x1k, x1M, MAX
- エネルギーコスト: 10倍の乗数適用
- 合成時に余剰粒子を返還

### 実績システム
- **元素実績**: 各元素1個保有 / 1mol保有
- **周期実績**: 各周期の元素コンプリート
- **族実績**: 各族の元素コンプリート
- **特殊実績**: ランタノイド/アクチノイドコンプリート
- **クリック実績**: 1k〜1Eクリック
- **プレイ時間実績**: 1分〜1千年紀
- **粒子実績**: 電子/陽子/中性子 1k〜1Y個保有

### プレステージ（Re-universe）
- 総獲得エネルギーのlog₁₀に応じた倍率
- リセット後も実績・総獲得エネルギー・プレイ時間・クリック数は保持
- 倍率は全てのエネルギー獲得に適用
